# Hamro Chautari Admin Panel

Next.js admin dashboard for managing the Hamro Chautari platform.

## Quick Start

```bash
cd admin_panel
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser.

## Features

- User management
- Content moderation  
- Analytics dashboard
- System settings

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint
